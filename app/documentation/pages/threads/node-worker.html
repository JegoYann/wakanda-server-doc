Node Worker

Why

  • You need to execute some code in background.
  • You need to do it in a non blocking way.
  • You need to use node modules.

What is it

  • A node worker is running in its own thread.
  • A node worker runs asynchronously.
  • A node worker provides a interface for messaging.
  • A node worker runs Node 6.9 LTS.

How to

The folder PROJECT/backend inside your Wakanda project is where you usually use a node worker.

It is composed of:

  • A worker running in its own thread.
  • A proxy that helps you to communicate with the worker.
