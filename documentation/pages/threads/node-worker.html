Node Worker

Why

  • You need to execute some code in background.
  • You need to do it in a non blocking way.
  • You need to use node modules.

What is it

  • A node worker is running in its own thread.
  • A node worker runs asynchronously.
  • A node worker provides an interface for messaging.
  • A node worker runs Node 6.9 LTS.

How to

You can require a:

  • Wakanda modules (defined in PROJECT/modules)
  • Node modules (defined in PROJECT/node_modules).

You can install a node module:

  • Right click on your BACKEND-PROJECT in the tree
  • Click on open in terminal
  • Type: $ npm install NodeModuleToInstall
